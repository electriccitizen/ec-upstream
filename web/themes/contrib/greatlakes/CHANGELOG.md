# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## 1.0.5

### Added
- Layout-dynamic CSS override to handle negative margin interactions with fixed hero positioning

### Changed
- GL-hero component updated for Canvas page builder compatibility
- GL-hero-cta component schema updated to work with Canvas
- Hero block template aligned with gl-hero component changes
- Page template updated to support fixed hero positioning
- Region CSS updated for improved hero layout handling

### Fixed
- Search form block configuration corrected
- Layout-dynamic negative bottom margin removed when fixed hero is present
- GL-hero component example data corrected

### Removed
- GL-hero example template file removed

## [1.0.4] - 2025-10-07

- Updated gl-header component template for improved property handling
- Enhanced header-popover component schema structure

### Fixed

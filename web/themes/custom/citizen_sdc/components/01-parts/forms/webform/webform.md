# Webform Component 

## Custom Variables
The webform component uses a few custom Twig variables in both webform.twig and the webform.html.twig that inherits it.  These custom variables are:
- webform_placer
- show_placer_title

The webform placer and show placer title variables come from the /includes/citizen_sdc.paragraph.inc and only apply if the webform is used in a form placer paragraph.
